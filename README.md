手が出せるOSSを目指します。

Issueは報告・質問・雑談なんでもどうぞ
読んだら読んだと書き込みます

## Pull Request
どんなPull Requestでもありがたいです。

- Code
- Comment
- fix typo
- テストの追加
- 翻訳(英語以外も)
- その他何でも

## お願い

英語・もしくは日本語でお願いします。

文字コードはutf-8
